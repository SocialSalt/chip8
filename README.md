# CHIP8 Emulator

This project is a chip8 emulator written in golang. It uses SDL2 for event polling and window mangement.
Special thanks to Austin Morlan for [their guide]( https://austinmorlan.com/posts/chip8_emulator/#16-bit-program-counter )
and to Matthew Mikolay for [their resources]( https://github.com/mattmikolay/chip-8/wiki/Mastering-CHIP%E2%80%908 )
and to Tim Franssen for providing [test roms]( https://github.com/Timendus/chip8-test-suite ).
